## IP Addresses & Ports

- Every device on a network has an **IP address** (like a home address)
- Ports identify **services** on a device
- Common ports:
  - SSH → 22
  - HTTP → 80
  - HTTPS → 443

**Practice Questions:**
1. Valid IPv4 address? → 192.168.1.1
2. HTTPS default port? → 443
3. Port 22 represents which service? → SSH

---

## DNS (Domain Name System)

- DNS translates **domain names → IP addresses**
- Common record types:
  - A → IPv4
  - AAAA → IPv6
  - CNAME → Alias
  - MX → Mail server

**Practice Questions:**
1. DNS translates domain names → IP addresses ✅
2. AAAA record maps domain → IPv6 ✅

---

## Firewalls

- Firewalls control network traffic (like a bouncer at a club)
- Types: Hardware & Software
- Example rules:
  - Allow port 22 → SSH allowed
  - Block port 21 → FTP blocked

**Practice Questions:**
1. Which firewall rule allows SSH? → Allow port 22 ✅
